# n8n Blog Automation Workflow for Mantra Pooja

This workflow automates the creation of high-quality, SEO-optimized blog posts for the Mantra Pooja website. It leverages Google Trends for topic discovery, Gemini AI for content and image generation, and the Mantra Pooja API for publishing.

## Workflow Overview

The process follows these steps:
1.  **Trigger**: Runs automatically every 12 hours (or manually).
2.  **Topic Sourcing**: Fetches keywords from a Google Sheet.
3.  **Trend Analysis**: Checks Google Trends for the selected keyword.
4.  **Content Generation**: Uses an AI Agent (Gemini) to write a detailed, structured blog post and generate an image prompt.
5.  **Image Generation**: Creates a featured image using Gemini.
6.  **Publishing**:
    *   Uploads the generated image to the server.
    *   Posts the final blog content (title, body, meta tags, image URL) to the Mantra Pooja API.

## detailed Configuration

### 1. Google Sheets (Source)
*   **Node**: `Get row(s) in sheet`
*   **Purpose**: Retrieves a list of blog topics/keywords.
*   **Setup**: Ensure you have a Google Sheet with a column named `keyword`.

### 2. Google Trends
*   **Node**: `Google_trends search`
*   **Purpose**: Validates interest and gathers trend data for the keyword.

### 3. AI Agent (The Writer)
*   **Node**: `AI Agent` with `Google Gemini Chat Model`
*   **Prompt**: Acts as a professional content strategist.
*   **Output**: A structured JSON object containing:
    *   `blog_title`
    *   `blog_content` (Introduction, Sections, FAQ, Conclusion)
    *   `image_prompt`
    *   `meta_title`, `meta_description`, `meta_tags`

### 4. Image Generation
*   **Node**: `Generate an image`
*   **Model**: `models/gemini-3-pro-image-preview`
*   **Input**: Uses the `image_prompt` generated by the AI Agent.

### 5. Deployment / Publishing
*   **Node**: `Image_Upload`
    *   **URL**: `http://YOUR_DOMAIN/api/webhook/upload-image`
    *   **Method**: `POST`
    *   **Body**: Binary data of the generated image.
*   **Node**: `HTTP Request` (Final Publish)
    *   **URL**: `http://YOUR_DOMAIN/api/blogs`
    *   **Method**: `POST`
    *   **Headers**: `x-webhook-secret: your_n8n_secret_here`
    *   **Body**:
        ```json
        {
          "title": "{{ $node['AI Agent'].json.output.blog_title }}",
          "content": {{ JSON.stringify($node['AI Agent'].json.output.blog_content) }},
          "slug": "{{ $node['AI Agent'].json.output.blog_title.toLowerCase().replace(/ /g, '-') }}",
          "image_url": "{{ $node['Image_Upload'].json.url }}",
          "tags": {{ JSON.stringify($node['AI Agent'].json.output.meta_tags) }},
          "published": true,
          "secret": "your_n8n_secret_here"
        }
        ```

## How to Test This Workflow in n8n

1.  **Import**: Copy the JSON provided and paste it into your n8n canvas (`Ctrl+V`).
2.  **Configure Credentials**:
    *   Connect your Google Sheets account.
    *   Connect your Google Gemini (PaLM) API account.
    *   Connect SerpApi (if using Google Trends node requiring it).
3.  **Set Environment Variables**:
    *   Update the API URLs in `Image_Upload` and `HTTP Request` nodes to point to your live server (e.g., `https://mantrapooja.com/api/...`).
    *   Update the `x-webhook-secret` to match the `N8N_WEBHOOK_SECRET` in your server's `.env.local`.
4.  **Run**: Click **"Test Workflow"** or manually execute the nodes step-by-step.

## Expected Result
*   A new blog post will appear in your Mantra Pooja Admin Dashboard and on the `/blogs` page.
*   The post will have a unique AI-generated cover image, SEO metadata, and structured content.
