# Project Requirements - Mantra-pooja Website

## Overview
A web platform to categorize and book Hindu Poojas, featuring a spiritual and premium design.

## Design Requirements
- **Theme**: Dark, spiritual, premium.
- **Colors**: Deep blues, blacks, glow effects (cyan/white), gold accents.
- **Typography**: Clean, modern sans-serif fonts (e.g., Inter, Roboto).

## Functional Requirements
### 1. Header
- Logo: "Design Scratch" (placeholder) -> "Mantra Pooja".
- Navigation: Home, Poojas, Festivals, Locations, Dealers, Blog.
- Action: "Book A Pooja" button.
- User: Language/Currency selector, Profile/Login.

### 2. Hero Section
- **Headline**: "Find the Right Pooja for Every Purpose".
- **Sub-headline**: "Discover authentic vedic poojas tailored to your spiritual needs...".
- **CTAs**: "Explore Poojas", "Learn About Us".
- **Visual**: Background image with a deity (Lord Shiva) and lighting effects.

### 3. Search & Filter
- **Functionality**: Filter Poojas by Purpose, Location, Occasion.
- **Search**: Text search bar for specific Poojas.

### 4. Popular Poojas
- **Display**: Grid layout.
- **Content**: Popular Poojas (e.g., Satyanarayan Pooja).
- **Card Details**: Icon/Image, Title, Short Description, "Read More" arrow.

### 5. Browse by Purpose
- **Display**: Grid/Carousel.
- **Content**: Categories of Poojas based on user intent (Health, Wealth, Peace, etc.).

## Technical Stack
- **Framework**: Next.js (App Router).
- **Language**: TypeScript.
- **Styling**: CSS Modules / Vanilla CSS.
- **Icons**: React Icons (or similar).
